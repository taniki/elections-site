## la composition de l'assemblée s'il n'y avait qu'un tour


```js
const composition_t1_max = (
	resultats_t1
	.transform(
		votes_to_resultats,
		resultats_to_composition
	)
	.join(
		composition_t2
		.derive({
			reel: d => d.sieges
		})
		.select('CodNua','reel')
	)
	.derive({
		diff: d => d.reel - d.sieges
	})
)

const majorite_t1_max = composition_t1_max.orderby(aq.desc('sieges')).slice(0, 1)
```
